Intro

Installable via npm install --save interface-discovery, it can also be used directly in the browser.

Download

The source is available for download from GitHub. Alternatively, you can install using npm:

$ npm install --save interface-discovery

You can then require() interface-discovery as normal:

const interfaceDiscovery = require('interface-discovery')

In the Browser

Interface-discovery should work in any ES2015 environment out of the box.

Usage:

<script type="text/javascript" src="index.js"></script>

The portable versions of interface-discovery, including index.js and index.min.js, are included in the /dist folder. Interface-discovery can also be found on unpkg.com under

  • https://unpkg.com/interface-discovery/dist/index.min.js
  • https://unpkg.com/interface-discovery/dist/index.js
